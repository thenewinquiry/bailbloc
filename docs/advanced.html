Windows with AMD GPUs

The tool of choice in this scenario is Claymore's miner, which can be found here (yes it's a forum post!) Check out that link for setup instructions, but basically download the binary (direct download here for convenience) and type the following commands in terminal:

setx GPU_FORCE_64BIT_PTR 0
setx GPU_MAX_HEAP_SIZE 100
setx GPU_USE_SYNC_OBJECTS 1
setx GPU_MAX_ALLOC_PERCENT 100
setx GPU_SINGLE_ALLOC_PERCENT 100
NsGpuCNMiner -o stratum+tcp://mine.xmrpool.net:5555 -u 442uGwAdS8c3mS46h6b7KMPQiJcdqmLjjbuetpCfSKzcgv4S56ASPdvXdySiMizGTJ56ScZUyugpSeV6hx19QohZTmjuWiM -p workerbee:bailbloc@thenewinquiry.com

Mac OS X with AMD GPUs

Check out xmrig-amd, install libmicrohttpd and libuv, build with cmake and make, then run:
./xmrig-amd --url mine.xmrpool.net:5555 --user 442uGwAdS8c3mS46h6b7KMPQiJcdqmLjjbuetpCfSKzcgv4S56ASPdvXdySiMizGTJ56ScZUyugpSeV6hx19QohZTmjuWiM --pass workerbee:bailbloc@thenewinquiry.com --keepalive --opencl-devices=1 --opencl-launch=256x8
